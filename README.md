🛡️ Projeto: Redesign da Jornada de Batalha Naval
📝 Visão Geral do Projeto
Este projeto foi desenvolvido para a disciplina de Interface e Jornada do Usuário, com o objetivo de reinterpretar o clássico jogo de tabuleiro Batalha Naval sob a ótica do design centrado no usuário. O foco principal foi mapear a experiência do jogador, identificar pontos de fricção no fluxo tradicional e criar uma interface digital intuitiva, imersiva e responsiva, que mantenha a tensão estratégica do jogo original de forma fluida.

🎯 Objetivos Principais
Otimizar a Jornada do Usuário (User Journey): Simplificar o fluxo desde o momento em que o jogador entra na sala até o posicionamento dos navios e os turnos de ataque.

Interface Limpa e Imersiva: Desenvolver uma UI que equilibre a estética militar/naval com a clareza visual, garantindo que o usuário entenda o status do tabuleiro (acertos, erros e frotas destruídas) num relance.

Acessibilidade e Feedback Visual: Implementar feedbacks claros (visuais e sonoros) para cada ação do jogador, reduzindo a carga cognitiva e eliminando a ambiguidade das jogadas.

🗺️ Etapas de Desenvolvimento (UX/UI)
Mapeamento da Jornada do Usuário: Construção do fluxo de tarefas (User Flow) para entender os momentos críticos do jogo, como a frustração de errar um tiro ou a satisfação de afundar uma embarcação.

Wireframing: Criação de protótipos de baixa e média fidelidade para testar a disposição dos tabuleiros (aliado vs. inimigo) e a ergonomia dos botões de ação.

Visual Design (UI): Definição de uma paleta de cores focada em tons escuros e contrastes neon [ou as cores que você usou, ex: tons de azul marinho e elementos de radar] para simular uma tela de comando tático.

Prototipagem Alta Fidelidade: Criação de interações dinâmicas (como o arrastar e soltar dos navios na fase de preparação) para simular a experiência real do usuário.

🚀 Diferenciais da Interface Desenvolvida
Modo de Posicionamento Inteligente: Sistema simplificado para girar e posicionar a frota, evitando cliques acidentais.

Histórico de Ataques Visual: Tela de radar intuitiva que diferencia claramente tiros na água de navios atingidos.

Dashboard de Status: Painel lateral que mostra a saúde da frota aliada e estimativas sobre os navios restantes do adversário.
